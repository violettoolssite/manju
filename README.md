# AI Storyboard Generator (AI 分镜创作平台)

这是一个基于 React 与前端 AI API 集成构建的专业影视分镜与视频创作平台。用户可以通过输入纯文本剧本，全自动完成剧本解析、角色与环境的 3D 资产设定、分镜图像生成以及最终的连贯视频生成。

## 核心功能

* 智能剧本解析：通过大语言模型自动从文本中提取出场人物、场景环境，并将其划分为连续的分镜脚本。
* 全局视觉风格与配置：内置 30 种以上的高质量视觉风格（涵盖写实、二次元、国风、赛博朋克等），支持自定义画面比例（16:9, 9:16, 1:1 等）与单双格排版布局。
* 严格的工作流控制：引入“解析 -> 三视图生成 -> 分镜图生成 -> 视频生成”的四步严谨审核工作流，每一步均支持人工确认与单点重新生成。
* 强约束与连贯性：底层提示词注入强约束机制，确保生成的图像与视频中不包含字幕水印，防止人物特征突变、衣着不一致、穿模或闪烁。
* 自定义模型提供商：支持用户在设置中自定义配置文本解析、图像生成、视频生成的 API 节点与密钥，并支持通过 API 实时拉取最新可用模型列表。
* 本地数据存储：所有项目资产与配置均通过浏览器本地存储（LocalStorage）进行安全保存。

## 技术栈

* 核心框架：React 18 (TypeScript)
* 构建工具：Vite
* 样式方案：Tailwind CSS
* 状态管理：Zustand
* 动画库：Framer Motion
* 图标库：Lucide React

## 快速开始

### 1. 环境准备

确保您的本地计算机已安装 Node.js (推荐版本 18 以上) 和 npm 或 pnpm。

### 2. 安装依赖

在项目根目录下执行以下命令安装项目所需的全部依赖：

```bash
npm install
# 或者使用 pnpm
pnpm install
```

### 3. 启动开发服务器

执行以下命令启动本地开发环境：

```bash
npm run dev
# 或者使用 pnpm
pnpm run dev
```

启动成功后，在浏览器中访问 http://localhost:5173 即可打开应用。

### 4. 构建生产环境

执行以下命令将项目打包为用于生产环境的静态文件：

```bash
npm run build
# 或者使用 pnpm
pnpm run build
```

构建完成后的文件将存放在 `dist` 目录中。

## 部署说明

本项目是一个纯前端单页应用（SPA），可以通过以下常见平台轻松部署：

### 部署到 Vercel (推荐)
1. 将项目推送到 GitHub 仓库。
2. 登录 Vercel，点击 Add New Project 并选择该 GitHub 仓库。
3. 在 Framework Preset 中选择 Vite。
4. 点击 Deploy，等待构建完成即可访问。
注：本项目已配置 `vercel.json` 路由重写规则以及 `.npmrc` 以适配 pnpm v10 在 Vercel 的权限要求。

### 部署到 Nginx
将打包后的 `dist` 文件夹上传到服务器，并在 Nginx 配置中添加路由重定向以支持单页应用：

```nginx
location / {
    try_files $uri $uri/ /index.html;
}
```

## 注意事项

使用本平台前，请务必在界面的“设置”面板中填入有效的第三方大模型 API Base URL 和 API Key（包含文本分析、图像生成、视频生成）。未配置 API 凭证将导致解析和生成功能无法正常工作。
